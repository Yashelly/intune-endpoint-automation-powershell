## Endpoint Automation with PowerShell

This repository contains small PowerShell scripts created to automate
routine endpoint and identity management tasks in enterprise Modern Workplace environments.

### Focus areas
- Microsoft Intune device compliance and configuration reporting
- Entra ID device and group hygiene
- Endpoint lifecycle and operational cleanup
- Basic administrative automation using PowerShell

### Scope
Scripts are intentionally simple and focused on operational efficiency.
They are not intended to represent a full automation framework.

### Disclaimer
All scripts are anonymized and simplified for demonstration purposes.
No production data or proprietary logic is included.
